DOCTYPE tells the browser the version of HTML that you are using. All modern HTML documents use HTML5 which is defined by line 1. The html tag wraps all of the other tags. all of your html will live inside of this tag, but there are 2 tags that valid as direct children, head and body. Lines 2 and 12. The head tag contains info about the doc and other resources that need to be pulled into the doc like CSS files, scripts, the page title, and more. The head tag does not contain visual elements for the user. Lines 3 and 7. The body tag contains all of the visual elements will be on the page like buttons, textboxes, images, etc. Lines 9 and 11.
